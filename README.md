## 使用

不通场景下的脚本，需要简单修改脚本内容，也就是代码逻辑变量，一般放在脚本文件顶部区域（有注释），比如数据库备份脚本，就需要修改默认要备份的数据库，以及主机、用户名、密码等信息。

每个脚本都可以通过 `--help`、`-help`、`-h` 可选命令来查看帮助说明。
