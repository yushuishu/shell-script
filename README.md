# shell-script

## 总结常用技巧

> 自定义颜色文本


红色文本：

```shell
echo -e "\033[31m红色文本\033[0m"
```


绿色文本：

```shell
echo -e "\033[32m绿色文本\033[0m"
```


黄色文本：

```shell
echo -e "\033[33m黄色文本\033[0m"
```


蓝色文本：

```shell
echo -e "\033[34m蓝色文本\033[0m"
```


紫色文本：

```shell
echo -e "\033[35m紫色文本\033[0m"
```


青色文本：

```shell
echo -e "\033[36m青色文本\033[0m"
```


白色文本：

```shell
echo -e "\033[37m白色文本\033[0m"
```
